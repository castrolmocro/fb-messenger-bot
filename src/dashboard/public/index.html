⬆️ رفع كوكيز جديدة
📋 صيغة JSON — مصفوفة من الكوكيز مثل c3c / Cookie-Editor
[{"key":"c_user","value":"123..."},{"key":"xs","value":"abc..."},...]
أو الصيغة الجديدة: [{"name":"c_user","value":"123..."}]
🔤 نص الكوكيز — انسخ كوكيز من المتصفح مباشرة
c_user=123456; xs=abc:def; datr=xxx; sb=yyy
يجب أن تحتوي على c_user وxs على الأقل
🔑 توكن Facebook — توكن يبدأ بـ EAAAA
EAAAA6z...لسنا مسؤولين عن صحة التوكن
سيتم تحويل التوكن تلقائياً إلى كوكيز صالحة
📖 كيف تحصل على كوكيز كاملة مع m_sess؟
⚠️ بدون m_sess لن يستقبل البوت رسائل!
الطريقة 1 — c3c Cookie Manager:
1. افتح messenger.com في المتصفح وسجّل دخول
2. افتح facebook.com في نفس المتصفح
3. استخدم c3c أو Cookie-Editor لتصدير كوكيز كلا الموقعين
4. ادمجهما في مصفوفة JSON واحدة وارفعها هنا
5. تأكد أن المصفوفة تحتوي على: c_user, xs, m_sess, datr
الطريقة 2 — نص الكوكيز المباشر:
1. افتح DevTools (F12) على messenger.com
2. اذهب إلى Network → أي طلب → Headers → Request Headers
3. انسخ قيمة Cookie: بالكامل
4. الصقها في تبويب "نص" أعلاه
